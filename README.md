# 효원(曉原) 미식회 🍜☕  
부산대 근처 맛집·카페 탐방 — 웹 데이터 기반 추천 웹페이지

> 웹상의 데이터를 수집(댓글/리뷰/평점 등)하여 부산대 근처의 맛집과 카페를  
> 평점과 리뷰 평가가 높은 순으로 선정하고, 웹사이트 형태로 제공하는 프로젝트입니다.

---

## 1. 프로젝트 개요

- **프로젝트 제목**: 부산대 근처의 맛집과 카페 탐방  
- **목표**: 부산대 학생 및 방문객이 정보의 홍수 속에서 빠르게 선택할 수 있도록  
  **추천 맛집·카페 정보를 정리·재해석해 웹페이지로 제공**
- **주요 타깃**: 대학생, 일반 시민

---

## 2. 주제 선정 이유

맛집/카페를 찾을 때 다양한 정보가 존재하지만, 오히려 선택이 어려워지는 경우가 많습니다.  
본 프로젝트는 웹상의 정보를 종합·정리해 **필요한 정보만 빠르게 검색**할 수 있도록 하여  
바쁜 일상 속에서 **시간을 단축**하고 선택 부담을 줄이는 것을 목표로 합니다.

---

## 3. 핵심 질문 (Research Questions)

## 5. 분석 방법(안)

기획서의 “리뷰·평점 평가” 방향을 기반으로, 아래 절차로 정리합니다.

### 5.1 분석 파이프라인
1. **수집**: 댓글/리뷰/평점/리뷰 수/작성일/링크 등 확보  
2. **정제**: 중복 제거, 결측 처리, 텍스트 정리(이모지/특수문자 등)  
3. **평점 기반 정렬**: 평점 + 리뷰 수를 고려한 신뢰도 반영(가중치 적용 가능)  
4. **리뷰 텍스트 요약/키워드**: 핵심 키워드 추출(메뉴/분위기/가성비 등)  
5. **추천 결과 생성**: 카테고리별 Top-K 추천 리스트 + 근거(키워드/요약) 제공

### 5.2 (권장) 선정 기준 예시
- 평점(주요 플랫폼) + 리뷰 수(신뢰도) 반영
- “부산대 인근” 범위(반경/행정구역) 설정
- 최신성(최근 N개월 리뷰 우선) 옵션
